อุปกรณ์ IoT ที่ใช้ทำชิ้นงาน

ระบบไฟห้องน้ำเปิด-ปิดอัตโนมัติ

ESP32

ESP32 คือ wifi microcontroller ที่ถูกพัฒนาต่อจาก ESP8266 โดยเพิ่ม CPU เป็น 2 core, Wi-Fi ที่เร็วขึ้น, มีขา GPIO ให้ใช้งานมากขึ้น และรองรับ Bluetooth อีกด้วย นอกจากนี้ ESP32 ยังมาพร้อมกับ touch-sensitive pins ที่สามารถใช้ปลุก ESP32 จากโหมด deep sleep และยังมี hall effect sensor และ temperature sensor ในตัว (รุ่นล่าสุดของ ESP32 ไม่มี temperature sensor ในตัวอีกแล้ว)

Relay

รีเลย์ (Relay) คือ สวิตช์ไฟฟ้าที่ใช้แม่เหล็กไฟฟ้าในการเปิดหรือปิดหน้าสัมผัสอย่างน้อยหนึ่งหน้าสัมผัส ซึ่งเป็นส่วนประกอบสำคัญในระบบไฟฟ้าและอิเล็กทรอนิกส์ มักใช้ในการควบคุมวงจรกำลังไฟสูงด้วยสัญญาณกำลังต่ำ

Solar cell

เป็นอุปกรณ์อิเล็กทรอนิกส์ที่ทำจากสารกิึ่งตัวนำ ที่มีความสามารถในการแปลงพลังงานแสงอาทิตย์ที่ตกกระทบตัวสารกิึ่งตัวนำเพื่อเปลี่ยนมาเป็นพลังงานไฟฟ้ากระแสตรง (DC current) ซึ่งเอานำมาใช้งานกับอุปกรณ์ไฟฟ้าทีเราได้ดำเนินการต่อกับระบบโซล่าเซลล์ของเราได้

solar inverter

อุปกรณ์แปลงพลังงานไฟฟ้าที่ได้จากแผงโซลาร์เซลล์ เพื่อส่งต่อพลังงานไฟฟ้าไปยังอุปกรณ์เครื่องใช้ไฟฟ้า ซึ่งการทำงานของอินเวอร์เตอร์โซลาร์เซลล์ เปรียบได้กับหม้อแปลงไฟฟ้า คือ มีหน้าที่แปลงพลังงานแสงอาทิตย์

เซ็นเซอร์จับการเคลื่อนไหว

“เครื่องตรวจจับความเคลื่อนไหว” คือ อุปกรณ์อิเล็กทรอนิกส์ที่ออกแบบมาเพื่อใช้ในการตรวจจับความเคลื่อนไหวทางกายภาพของสิ่งมีชีวิต โดยจะนิยมนำมาใช้ในระบบดูแลรักษาความปลอดภัยภายในบ้าน หรือในธุรกิจ เพื่อตรวจจับผู้บุกรุกเข้ามายังในสถานที่
